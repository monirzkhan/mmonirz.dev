## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Monirzkhan/mmonirz.dev.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
